SportBotX

SportBotX is a web-based application that allows users to register and log in to an account, select criteria, and view results. The SportBotX web application enables users to compare sport running shoes based on various attributes such as quality, price, weight, and design. It ensures that users receive useful comparisons for making informed decisions. It is to help users make wise decisions before buying running shoes. This application focuses on assisting SportBotX users in effectively comparing sport running shoes. SportBotX provides a solution for users who have difficulties choosing the right sport running shoes.
